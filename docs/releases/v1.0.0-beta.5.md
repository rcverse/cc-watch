# v1.0.0-beta.5

## What changed

- Added persistent session pins and Reminder selections so watched sessions
  remain loaded beyond the recent-session limit.
- Added attention ordering for active, unknown, and expired sessions, while
  preserving the selected session across refreshes and reordering.
- Added current model, context-token, and earlier-model details to Session Info.
- Explained unknown cache timing states, including cache resets, failed
  responses, and ambiguous cache tiers, with a safe next action.
- Refreshed the cache estimate after a confirmed KeepAlive send.
- Expanded the synthetic TUI demo and README documentation, including updated
  session-list and KeepAlive recordings.
